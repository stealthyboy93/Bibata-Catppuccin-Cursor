# 🎨 Catppuccin Bibata Cursors

Beautiful **Bibata** cursors recolored with every official **Catppuccin** flavor for Linux.


---

## ✨ About

Catppuccin Bibata is a collection of Linux **XCursor** themes inspired by the original Bibata cursor design and recolored using the official Catppuccin palettes.

Each flavor has been carefully crafted to preserve the recognizable Bibata appearance while blending seamlessly with Catppuccin themes, icons, terminals, and desktop environments.

Whether you're using a dark or light desktop, there's a matching cursor theme for your setup.

## 🎨 Available Flavors

| Flavor           | Description                                     |
| ---------------- | ----------------------------------------------- |
| 🌙 **Mocha**     | Rich, vibrant colors designed for dark desktops |
| 🌌 **Macchiato** | A balanced dark palette with cooler tones       |
| 🌆 **Frappé**    | Soft and muted colors for a relaxed appearance  |
| ☀️ **Latte**     | Bright and elegant colors for light themes      |

---

## 📦 Features

* 🎨 All four official Catppuccin flavors
* 🖱️ Based on the original Bibata cursor design
* 🐧 Native Linux XCursor themes
* ⚡ Compatible with both X11 and Wayland
* 💻 Works with GNOME, KDE Plasma, Hyprland, Sway, XFCE, Cinnamon, MATE, LXQt, i3, bspwm, Openbox, and many more

---

## 💻 Compatibility

Catppuccin Bibata works anywhere the Linux **XCursor** specification is supported, including:

* GNOME
* KDE Plasma
* Hyprland
* Sway
* XFCE
* Cinnamon
* MATE
* LXQt
* i3
* bspwm
* Openbox
* And many other X11 and Wayland environments
