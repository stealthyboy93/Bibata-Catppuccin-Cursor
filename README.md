## ✨ About

Catppuccin Bibata brings the complete Catppuccin color palette to one of Linux's most popular cursor themes.

Every cursor has been carefully recolored while preserving the familiar Bibata design, creating a consistent experience across your desktop, GTK theme, icons, terminal, and applications.

Whether you prefer dark or light desktops, there's a flavor that fits perfectly.

## 🎨 Available Flavors

| Flavor           | Description                                     |
| ---------------- | ----------------------------------------------- |
| 🌙 **Mocha**     | Rich, vibrant colors designed for dark desktops |
| 🌌 **Macchiato** | A balanced dark palette with cooler tones       |
| 🌆 **Frappé**    | Soft, muted colors for a subtle appearance      |
| ☀️ **Latte**     | Clean and elegant colors for light themes       |

---

## ✨ Features

* 🎨 All four official Catppuccin flavors
* 🖱️ Based on the elegant Bibata cursor design
* 🐧 Native Linux XCursor themes
* ⚡ Compatible with both X11 and Wayland
* 💻 Works with GNOME, KDE Plasma, Hyprland, Sway, XFCE, Cinnamon, MATE, LXQt, and more

## 💻 Compatibility

These cursor themes work with any desktop environment that supports the Linux XCursor specification, including:

* GNOME
* KDE Plasma
* Hyprland
* Sway
* XFCE
* Cinnamon
* MATE
* LXQt
* i3
* bspwm
* Openbox
